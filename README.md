# AutoCare Pro — Automotive One-Stop Service PWA

A mobile-first Progressive Web App (PWA) frontend demo for an automotive one-stop service business. Built with Next.js 14, TypeScript, Tailwind CSS, Shadcn/UI, and Framer Motion.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Mobile-first responsive design
- Bilingual support (English / 中文)
- Product browsing with search & filtering
- Shopping cart with quantity controls
- Mock checkout with payment UI
- Loyalty points & redemption system
- User profile with order history
- PWA installable (Add to Home Screen)

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Shadcn/UI**
- **Framer Motion**
- **next-pwa**

## Deploy

Deploy target: [Vercel](https://vercel.com)

```bash
npm run build
```
